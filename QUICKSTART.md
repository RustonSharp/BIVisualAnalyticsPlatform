# 快速开始指南

## 启动应用

1. 确保已安装所有依赖：
```bash
pip install -r requirements.txt
```

2. 运行应用：
```bash
python app.py
```

3. 在浏览器中打开：
```
http://localhost:8050
```

## 界面导航

### 1. 数据源管理
- 点击左侧导航栏的"数据源管理"
- 点击"+ 新增数据源"按钮
- 选择数据源类型：
  - **本地文件**：拖拽或选择 CSV/Excel 文件
  - **数据库**：填写 PostgreSQL/MySQL 连接信息
  - **REST API**：输入 API 地址和参数
- 点击"测试连接"验证配置
- 点击"保存"完成配置

### 2. 图表设计器
- 点击左侧导航栏的"图表设计器"
- 选择数据源
- 从字段列表拖拽字段到配置区（X 轴、Y 轴、分组）
- 选择图表类型（折线图、柱状图、饼图等）
- 配置聚合函数和样式
- 在右侧查看图表预览

### 3. 仪表盘
- 点击左侧导航栏的"仪表盘"
- 使用顶部的时间筛选器过滤数据
- 查看多个图表的综合分析
- 使用导出功能导出为 PNG/PDF/HTML

### 4. 设置
- 配置自动刷新间隔
- 设置默认图表主题
- 导入/导出数据源配置

## 注意事项

⚠️ **重要提示**：
- 此版本**仅提供 UI 界面**，不包含实际的数据处理逻辑
- 所有按钮和操作仅用于界面展示，不会实际执行功能
- 图表预览使用示例数据展示
