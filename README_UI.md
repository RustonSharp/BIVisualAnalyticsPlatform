# BI 数据可视化与分析平台 - UI 界面

这是一个基于 Plotly Dash 的 BI 数据可视化平台 UI 界面，**仅提供 UI 界面，不包含实际操作逻辑**。

## 功能特性

### 1. 数据源管理页面
- 支持三种数据源类型：
  - **本地文件**：CSV / Excel 文件上传（拖拽上传）
  - **数据库连接**：PostgreSQL / MySQL 配置
  - **REST API**：通过 URL 和参数配置 API 数据源
- 数据源列表展示（名称、类型、状态、最后更新时间）
- 数据预览功能

### 2. 图表设计器页面
- 左侧配置区域：
  - 数据源选择
  - 字段列表（可拖拽）
  - 图表类型选择（折线图、柱状图、饼图、表格、组合图）
  - 拖拽式配置区（X 轴、Y 轴、分组/颜色）
  - 聚合函数选择
  - 样式配置（标题、颜色主题、显示选项）
- 右侧预览区域：
  - 实时图表预览
  - 保存和导出功能

### 3. 仪表盘页面
- 顶部工具栏（新建、编辑、导出）
- 时间范围筛选器（快捷选项 + 自定义日期范围）
- 网格布局展示多个图表
- 示例图表展示（折线图、柱状图、饼图、表格）

### 4. 设置页面
- 通用设置（自动刷新间隔、默认图表主题）
- 数据源配置管理（导入/导出配置）

## 技术栈

- **Dash**：Web 应用框架
- **Dash Bootstrap Components**：UI 组件库
- **Plotly**：图表可视化库
- **Pandas**：数据处理库

## 安装依赖

```bash
pip install -r requirements.txt
```

## 运行应用

```bash
python app.py
```

应用将在 `http://localhost:8050` 启动。

## 界面说明

### 导航栏
左侧固定导航栏包含以下页面：
- 📊 数据源管理
- 📈 图表设计器
- 📉 仪表盘
- ⚙️ 设置

### 数据源管理
1. 点击"新增数据源"按钮
2. 选择数据源类型（文件/数据库/API）
3. 填写相应配置信息
4. 点击"测试连接"验证
5. 点击"保存"完成配置

### 图表设计器
1. 选择数据源
2. 从字段列表拖拽字段到配置区
3. 选择图表类型
4. 配置聚合函数和样式
5. 查看右侧预览效果

### 仪表盘
- 使用时间筛选器过滤数据
- 查看多个图表的综合分析
- 支持导出为 PNG/PDF/HTML

## 注意事项

⚠️ **重要提示**：
- 此版本仅为 UI 界面展示，**不包含实际的数据处理逻辑**
- 所有按钮和操作仅展示界面，不会实际执行功能
- 图表预览使用示例数据进行展示
- 数据源配置表单仅展示，不会实际保存或连接

## 后续开发建议

要实现完整功能，需要开发以下模块：
1. 数据源适配器（文件上传、数据库连接、API 请求）
2. 数据处理引擎（字段类型识别、数据聚合）
3. 图表生成引擎（根据配置生成 Plotly 图表）
4. 配置管理器（保存/加载数据源和仪表盘配置）
5. 导出功能（PNG/PDF/HTML 导出）
